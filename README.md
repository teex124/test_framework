## 📖 О проекте

<p align="center">
  <img src="https://user-images.githubusercontent.com/74038190/212284087-bbe7e430-757e-4901-90bf-4cd2ce3ca185.gif" width="400" align="right">
</p>

Этот проект — демонстрация современного подхода к автоматизации тестирования веб-приложений. Фреймворк построен с использованием **Page Object Model (POM)** для обеспечения читаемости и лёгкой поддержки тестов.

**Ключевые возможности:**
- ✅ **Playwright** — быстрые и надёжные UI-тесты
- ✅ **Allure Reports** — детальная аналитика с историей и трендами
- ✅ **GitHub Actions** — полный CI/CD пайплайн
- ✅ **Telegram-уведомления** — мгновенные отчёты о результатах
- ✅ **GitHub Pages** — публичный хостинг отчётов
